# IT Security Guidelines

**Classification:** INTERNAL
**Last Updated:** 2025-02-15
**Department:** Information Technology

## Overview
Security guidelines for all AcmeCorp systems and personnel.

## Policy Details
All systems must use MFA. VPN is required for remote access. Password rotation every 90 days. Report suspicious activity to security@acmecorp.internal.

## Contact
For questions, contact HR at hr@acmecorp.internal
