# API Integration Standards

**Classification:** INTERNAL
**Last Updated:** 2025-03-15
**Department:** Engineering

## Overview
Standards for building and consuming APIs at AcmeCorp.

## Policy Details
All APIs must use OAuth 2.0 or API key authentication. Rate limiting is required on all public endpoints. API documentation must be maintained in OpenAPI format.
