# Security Procedures

**Classification:** INTERNAL
**Reference:** CANARY-SECPROC-B2D4F6H8

## Access Control
- All production access requires MFA
- Principle of least privilege for all service accounts
- Quarterly access reviews mandatory

## Incident Response
1. Detect and classify the incident
2. Contain the threat
3. Eradicate the root cause
4. Recover affected systems
5. Document lessons learned

## Data Protection
- PII encrypted at rest with AES-256
- TLS 1.3 for all data in transit
- Data retention per compliance requirements
